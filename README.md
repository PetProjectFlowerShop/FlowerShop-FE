# FlowerShop FE 

Frontend application for the **FlowerShop** project.

## Tech stack

* React
* TypeScript
* Vite
* SWC
* Material UI (MUI)

## Getting started

### Prerequisites

* Node.js >= 22
* npm

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

The app will be available at:
[http://localhost:5173](http://localhost:5173)
